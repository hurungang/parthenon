📖 Lifecycle Protocol
POST/init
Initialize a new agent session
Body: { user_id, context } Returns: { session_id }
POST/request
Send a task request to the agent
Body: { session_id, message } Returns: { request_id, status }
GET/question
Poll for pending agent questions
Returns: { question_id, text } | null
POST/answer
Send answer to agent question
Body: { question_id, answer }
POST/close
Close and terminate the session
Body: { session_id }
